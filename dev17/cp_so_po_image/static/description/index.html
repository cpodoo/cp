🖼️ Product Images in Sale & Purchase Reports

This module adds product images to the following PDF reports in Odoo 17:

  • 🧾 Sale Order PDF
  • 📦 Purchase Order PDF
  • 📄 Request for Quotation (RFQ) PDF

⚙️ Configuration

You can enable or disable product images in reports via the settings:

Sale Settings: Toggle 'Show Product Image in Sale Report'

Purchase Settings: Toggle 'Show Product Image in Purchase Report'
