🏠 Property Management Module

This Odoo module helps manage properties, their types, offers, contractors, and payments in a streamlined manner.

📋 Key Features

  • Manage property listings with detailed attributes
  • Assign property types and custom tags
  • Create and track offers from potential buyers
  • Link contractors to properties for maintenance or projects
  • Manage and record payment history for each property
  • Automatic sequence numbers for records

📸 Property List View

The property list view displays all property records with tags, contact, budget, and client information in a clean format.

🧱 Technical Details

  • Models:
    • property_management
    • property_management_type
    • property_management_tag
    • property_management_offer
    • property_management_contractor
    • property_management_payment
  • Includes XML data for sequence generation

⚙️ Installation

  1. Download the module and place it in your custom addons folder
  2. Restart the Odoo server
  3. Activate developer mode
  4. Go to Apps → Update App List
  5. Search for Property Management and install

🧪 Tested on

  • Odoo 16
