Task Progress Restriction Setting

If user selects 'Restrict Task Progress Before All Checklist Activity Completion' option from project settings, a checkbox will become visible in Task Stages configuration.

Task Stage Restriction Configuration

Suppose user sets the 'Checklist Task Progress Restriction' checkbox for the 'Done' stage. Then, a task cannot be moved to 'Done' until all its checklist activities are either cancelled or completed. This setting can be applied to any task stage.

Checklist Activity Stages

Default activity stages are 1. TODO, 2. Completed, 3. Canceled. You can create up to 20 more custom stages with sequences between 2 and 20 (1, 21, and 22 are reserved).

Checklist Master Configuration

Define reusable checklists by adding a name, description, list of standard activities, and optionally linking them to a specific project. If linked to a project, they will only be available for tasks within that project.

Using Checklists in Project Tasks

Select a checklist from the 'Checklist' field on a task. A new 'Checklist' tab appears, automatically populated with the activities from the master checklist. Manage activities, view stages (with different colors), and see the overall checklist progress gauge.

Checklist Activity Detail View

View and manage individual checklist activities in a popup form view.

Checklist Progress in Task Kanban View

See the checklist completion progress directly on task Kanban cards.

Checklist Progress in Task List View

A dedicated column shows the checklist progress percentage in the task list view.

Checklist Activity Count in Project Kanban View

Quickly see the total number of checklist activities associated with each project on the project Kanban view.

Checklist Activity Count in Project Form View

A smart button on the project form view shows the total count of related checklist activities and allows direct navigation to them.

Dedicated Checklist Activity Menu

A separate menu item provides access to all checklist activities across projects. Includes useful filters (by Stage, Task, Project) and grouping options for efficient management and bulk operations.
