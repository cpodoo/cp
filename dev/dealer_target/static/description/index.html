Purpose & Highlights

The Dealer Sales Target Management module for Odoo 18.0 helps businesses set, track, and manage sales targets for dealers and salespersons. It enables real-time progress tracking, automated achievement calculations, and multiple sales milestone criteria (order confirmation, delivery, invoice payment). With list and form views, progress indicators, and email notifications, this module enhances sales performance visibility and streamlines operations.

Example: Sales Target Form (Draft Stage)

Updating Sales Target Achievement Criteria

Users can define how sales achievements are calculated through a menu offering criteria like "Sale Order Confirmed," "Delivery Order Done," "Invoice Created," and "Invoice Paid." The system then updates the progress dynamically for accurate performance evaluation.

Example: Achievement Criteria Configuration

Sales Target Overview Dashboard

A centralized dashboard provides a clear view of assigned sales targets, including start and end dates, target values, achieved sales, and progress indicators. Visual progress bars let managers monitor performance at a glance and make data-driven decisions.
