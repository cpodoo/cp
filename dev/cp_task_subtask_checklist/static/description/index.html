Project Task SubTask Checklist

With the help of this module you can divide task or sub task into list of activities, so task and subtask progress will easily control in Odoo project management.

Key Features

  • ✔ Any Number Of Configurable Checklists (General or Project Specific).
  • ✔ Configurable Checklist Stages (Default 3 Stages; you can add more stages).
  • ✔ Only Admin And Manager Can Add/Remove Checklists.
  • ✔ Any Number Of Checklist Activities per Task.
  • ✔ Set Color Based on Checklist Activity Stage (via view modifications if needed).
  • ✔ Separate Checklist Activity Menu With Different Filter/Group By Options.
  • ✔ Complete / Cancel Checklist Activities in Bulk.
  • ✔ Checklist Progress Gauge/Bar in Task Kanban, List, and Form Views.
  • ✔ Activities Can Be Checked/Managed From Task Form View.
  • ✔ Restrict Task Progress until all checklist items are Canceled or Completed.

Screenshots

Task Progress Restriction Setting

If user selects 'Restrict Task Progress Before All Checklist Activity Completion' from project settings, a checkbox will appear in Task Stages configuration.

Task Stage Restriction Configuration

If 'Checklist Task Progress Restriction' is enabled for the 'Done' stage, the task cannot be moved to 'Done' until all checklist activities are completed or cancelled.

Checklist Activity Stages

Default stages are TODO, Completed, Canceled. You can add up to 20 more stages.

Checklist Master Configuration

Define checklist name, description, and activities. Link to project if needed.

Using Checklists in Project Tasks

Select checklist from task form, and activities will be added automatically.

Checklist Activity Detail View

View and manage individual checklist activities in a popup form view.

Checklist Progress in Task Kanban View

See the checklist completion progress directly on task Kanban cards.

Checklist Progress in Task List View

A dedicated column shows the checklist progress percentage in the task list view.

Checklist Activity Count in Project Kanban View

Quickly see the total number of checklist activities associated with each project on the project Kanban view.

Dedicated Checklist Activity Menu

A separate menu item provides access to all checklist activities across projects. Includes useful filters (by Stage, Task, Project) and grouping options for efficient management and bulk operations.
