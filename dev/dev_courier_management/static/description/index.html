Courier Management

This odoo application adds Courier Management functionality into the odoo. If you are running a courier service then this odoo application will be very useful for you. Create Courier Request for the customer. Add sender and receiver details into the request(name, address, contact etc), add courier details(kilometer, distance charge, additional charge etc)into the request.Add parcel details into the request with quantity, weight, dimension and total charge of the courier will be calculated automatically. Add some parcel images into the request.Take digital signature of sender on the courier request. Define various stages of the request(e.g. collected, dispatched, delivered etc.) Create Customer Invoice for the courier request.Send courier request by email.Various kinds of reports are provided for better understanding
This odoo application adds website portal feature into Courier Management. Using this odoo application allow your customers to view their Courier Requests. Various kind of Sort By options provided. Customer can Print/Download all courier requests as PDF. Customer can see Signature on courier request from the website portal.Customer can see parcel images on courier request from the website portal.
This odoo application adds rating feature into the Courier Management. Send Courier Request rating email to the customer, so customer can rate your services. Feedback received from the customer will display on Courier Request
This odoo application adds Tracking functionality for the Courier Request. Using this odoo application customer can track their courier request through website portal. Separate menu provided on website for Courier Tracking. Customer can track courier request using unique number of the courier request.

Features

  • Advance Dashboard is provided for analysis.
  • Configure Stages for Courier Request.
  • Configure Contacts of courier service, Separate menu for contact is also created.
  • Configure Weight Price Rules.
  • Configure Dimension Price Rules.
  • Configure Distance Price Rules.
  • Create Courier Request with detailed information of the courier.
  • Create Invoice for the Courier Request.
  • Send Courier Request by email.
  • Reporting :
    • Print Courier Request as PDF Report
    • Print Courier Request History as PDF Report
    • View Courier Request Analysis
  • Customer can see all the Courier Requests from website.
  • Print/Download Courier Requests as PDF.
  • Sort Courier Requests by :
    • Number
    • Registration Date
  • Configure Rating Email Template into the stages of the Courier Request.
  • So based on Rating Email Template, rating email will be sent to the customer.
  • e.g. You have configured rating email template into the 'Delivered' stage of the Courier Request, So rating email will be sent automatically to the customer when Courier Request will move to the 'Delivered' stage.
  • Customer's rating and feedback will be visible on the Courier Request.
  • Note : Sender of the Courier Request considered as customer.
  • Track Courier Request from the website portal.
  • Separate menu provided on the website named Courier Tracking.
  • Track Courier Request using unique number of the Courier Request.
