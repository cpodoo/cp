📚 Library Management

The Library Management module for Odoo simplifies tracking books, members, and borrowing—all from an intuitive dashboard.

Perfect for schools, universities, and public libraries seeking effortless book management.

🔑 Key Features

  • Detailed cataloging: authors, ISBN, categories
  • Member profiles with borrowing history
  • Issuing, returning, automatic overdue tracking
  • Barcode & QR management
  • Custom reports for books and members

💼 Technical Details

  • Odoo 16/17 Community & Enterprise compatible
  • No external dependencies required
  • One-step menu setup for books, members & transactions
  • Easy upgradable with future releases

🛠️ Installation

  1. Copy to Odoo addons folder
  2. Update app list (Settings → Apps → Update)
  3. Search for Library Management and click Install
